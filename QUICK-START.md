## 5-Minute Setup

### 1. Log into WordPress
Go to: **shelzyperkins.com/wp-admin**

### 2. Install Theme
- Appearance → Themes → Add New → Upload Theme
- Choose `shelzyperkins-theme.zip`
- Install → Activate

### 3. Install 3 Must-Have Plugins
Go to Plugins → Add New, search and install:
1. **Rank Math SEO**
2. **LiteSpeed Cache**
3. **ShortPixel**

### 4. Create Your First Post
- Posts → Add New
- Copy content from `blog-posts/01-best-amazon-beauty-dupes.md`
- Add a featured image
- Publish!

### 5. You're Live!
Visit shelzyperkins.com to see your site.
