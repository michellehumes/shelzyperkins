## TASK 1: CREATE HEYGEN ACCOUNT & FIRST VIDEO

1. Go to https://heygen.com
2. Click "Sign Up" or "Get Started Free"
3. Create account (use user's email if provided, or create new)
4. Once logged in, click "Create Video"
5. Select avatar: Choose a young, friendly-looking female avatar (ages 20-30, casual style)
6. Set video format to 9:16 (TikTok vertical)
7. Set background to solid light pink (#FFE4EC) or white
8. Paste this script:

```
Hey! So everyone keeps asking where I get these personalized water bottles, and honestly, I make them myself.

I use this font called Pecita, which is totally free, size it to about 2 inches, and let my Cricut do the rest.

The vinyl is waterproof and dishwasher safe. I can make any name you want. DM me if you want one - they start at just $25!
```

9. Click Generate
10. Wait for video to render
11. Download the video
12. Save as "video-01-intro.mp4"
