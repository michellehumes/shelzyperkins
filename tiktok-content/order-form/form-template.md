# Order Form Template

Use this to set up your Google Form or Stan Store order form.

---

## Google Form Setup

### Form Title
```
Custom Personalized Water Bottle Order
```

### Form Description
```
Thanks for ordering! Please fill out all fields below.

Price: $25 per bottle (+ $5 shipping)
Processing time: 3-5 business days
Payment: Venmo/PayPal/CashApp (details sent after submission)
```

---

## Questions to Include

### Question 1: Name for Bottle
```
Type: Short answer
Question: Name for the bottle (exactly as you want it spelled)
Description: This is how it will appear on your bottle. Double-check spelling!
Required: Yes
```

### Question 2: Vinyl Color
```
Type: Multiple choice
Question: Choose your vinyl color
Options:
- Hot Pink
- Baby Pink
- Gold
- Rose Gold
- Teal
- Black
Required: Yes
```

### Question 3: Shipping Address
```
Type: Paragraph (long answer)
Question: Shipping address
Description: Please include: Street, City, State, ZIP
Required: Yes
```

### Question 4: Phone Number
```
Type: Short answer
Question: Phone number
Description: For shipping updates only
Required: Yes
```

### Question 5: Email
```
Type: Short answer
Question: Email address
Description: We'll send order confirmation and tracking here
Required: Yes
Validation: Must be email format
```

### Question 6: Special Requests (Optional)
```
Type: Paragraph
Question: Any special requests or notes?
Description: Font size preference, gift message, etc.
Required: No
```

## Confirmation Message

Set this as the confirmation message after submission:

```
Thank you for your order!

NEXT STEPS:
1. Send payment of $30 ($25 + $5 shipping) to:
   - Venmo: @YourVenmoHandle
   - PayPal: your@email.com
   - CashApp: $YourCashTag

2. Include your NAME in the payment note

3. You'll receive an email confirmation within 24 hours

4. Your bottle will ship within 3-5 business days

Questions? DM us on TikTok @YourHandle

Thank you for supporting a small business!
```

## Payment Options to Set Up

### Venmo
1. Download Venmo app
2. Create account
3. Get your @handle
4. Turn on business profile (optional)

### PayPal
1. Go to paypal.com
2. Sign up for free account
3. Use email for payments
4. Consider PayPal.me link for easier sharing

### CashApp
1. Download CashApp
2. Create account
3. Get your $Cashtag
4. Link bank account

## DM Response Templates

### When someone DMs "ORDER":
```
Hey! Thanks for your interest!

Here's how to order:
1. Fill out this form: [YOUR GOOGLE FORM LINK]
2. Send $30 to [YOUR PAYMENT INFO]
3. That's it! I'll start making your bottle

Current turnaround: 3-5 business days

Let me know if you have any questions!
```

### When someone asks "How much?":
```
$25 per bottle + $5 shipping = $30 total

You pick:
- Your name (any spelling)
- Vinyl color (hot pink, baby pink, gold, rose gold, teal, or black)

DM me when you're ready to order and I'll send you the form!
```

### When someone places an order:
```
Got your order! Thanks so much!

Just confirming:
- Name: [NAME]
- Color: [COLOR]
- Shipping to: [ADDRESS]

I'll get started on this today and send you a pic before shipping. Should be on its way within 3-5 days!
```

### When shipping:
```
Your bottle is on its way!

Tracking: [TRACKING NUMBER]

It should arrive in [X] days. Send me a pic when you get it - I'd love to see it!

Thanks again for your order!
```
