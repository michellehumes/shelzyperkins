## SEO Strategy

### On-Page SEO Basics
- Use focus keywords in titles and headers
- Write meta descriptions for all posts
- Use descriptive image alt text
- Internal link between related posts
- Use categories and tags strategically

### Content Strategy
- Publish 1-2 posts per week minimum
- Update posts annually (especially product roundups)
- Target long-tail keywords like:
  - "best Amazon finds under $30"
  - "Amazon organization products 2025"
  - "cheap Amazon travel essentials"

### Pinterest Strategy (Highly Recommended)
- Create 2-3 pins per blog post
- Use vertical images (2:3 ratio)
- Add text overlay with post title
- Pin consistently using Tailwind
