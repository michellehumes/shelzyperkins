Google Analytics 4 Setup for ShelzyPerkins.com

Step 1: Create GA4 Property

  1. Go to analytics.google.com
  2. Click Admin (gear icon)
  3. Click "Create Property"
  4. Property name: "ShelzyPerkins"
  5. Time zone: Your timezone
  6. Currency: USD
  7. Click Next and select "Other" for business type
  8. Select your business objectives (Generate leads, Drive sales)

Step 2: Data Stream Setup

  1. Select "Web" as platform
  2. Website URL: https://shelzyperkins.com
  3. Stream name: "ShelzyPerkins Web"
  4. Enable Enhanced Measurement
  5. Copy your Measurement ID (G-XXXXXXXXXX)

Step 3: Install Tracking Code

Option A: Add to Theme Customizer

Go to Appearance > Customize > ShelzyPerkins Settings > Analytics and enter your GA4 ID.

Option B: Manual Installation

Add this code to your header.php before </head>:


<!-- Google Analytics 4 -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX', {
    'cookie_flags': 'SameSite=None;Secure'
  });
</script>

Step 4: Configure Custom Events

Affiliate Link Clicks

The theme automatically tracks affiliate clicks. Configure the event in GA4:

  1. Go to Admin > Events
  2. Click "Create Event"
  3. Event name: affiliate_click
  4. Mark as conversion

Newsletter Signups

  1. Create event: newsletter_signup
  2. Mark as conversion

Step 5: Enhanced Ecommerce (Optional)

For advanced tracking, enable the following events:

  • view_item - When product card is viewed
  • select_item - When product is clicked
  • add_to_cart - When affiliate link is clicked

Step 6: Create Custom Reports

Affiliate Performance Report

  1. Go to Explore > Blank
  2. Dimensions: Event name, Page path, Link type
  3. Metrics: Event count, Users
  4. Filter: Event name = affiliate_click

Top Converting Pages

  1. Dimensions: Page path
  2. Metrics: Conversions, Sessions
  3. Sort by Conversion rate

Step 7: Set Up Goals

Goal Type Value
Affiliate Click Event $0.10
Newsletter Signup Event $1.00
Page Engagement Engagement $0.05

Step 8: Google Search Console Integration

  1. Go to Admin > Property Settings > Search Console Links
  2. Click "Link"
  3. Select your Search Console property
  4. Complete the linking process

Verification Checklist

  • [ ] Affiliate click events tracking
  • [ ] Newsletter signup events tracking
  • [ ] Search Console connected
