# Content Publishing Checklist

Use this checklist every time you publish a new blog post to ensure maximum profitability and SEO performance.

---

## Pre-Writing Phase

### Research & Planning
- [ ] **Keyword research completed**
  - Target keyword identified: _____________
  - Search volume checked (aim for 500+ monthly searches)
  - Competition assessed (aim for low-medium difficulty)

- [ ] **Product research completed**
  - 10+ products identified
  - ASINs collected and verified
  - Prices and ratings confirmed
  - Products currently in stock on Amazon

- [ ] **Competition analysis done**
  - Reviewed top 3 competing posts
  - Identified gaps to fill
  - Noted what makes their posts rank well

- [ ] **Post type decided**
  - [ ] Best of list
  - [ ] Gift guide
  - [ ] Product comparison
  - [ ] Deal roundup
  - [ ] How-to guide
  - [ ] Product review

---

## Writing Phase

### Content Structure
- [ ] **Compelling headline created**
  - Includes target keyword
  - Includes number (if list post)
  - Includes price point (if applicable)
  - Under 60 characters
  - Examples:
    * "15 Best Amazon Beauty Products Under $25"
    * "Kitchen Gadgets That Look Expensive But Aren't"
    * "Tech Gifts Under $50 That Everyone Loves"

- [ ] **Introduction (100-150 words)**
  - Hook in first sentence
  - Promise/benefit stated clearly
  - Target keyword mentioned in first 100 words
  - Personal connection or story

- [ ] **Table of Contents added** (for 2000+ word posts)
  - Links to H2 sections
  - Improves user experience
  - Increases time on page

- [ ] **Affiliate disclosure at top**
  - Used `[affiliate_disclosure]` shortcode
  - Placed before first affiliate link
  - Clear and compliant with FTC guidelines

### Main Content
- [ ] **Word count target met**
  - Minimum: 1500 words
  - Ideal: 2000-3000 words
  - Maximum: No limit, but stay focused

- [ ] **Subheadings optimized (H2s, H3s)**
  - At least 5 H2 subheadings
  - Include variations of target keyword
  - Clear and descriptive
  - Break up content every 300 words

- [ ] **Product descriptions written**
  - Each product has 100-150 word description
  - Includes unique benefits
  - Mentions specific features
  - Addresses buyer concerns
  - Personal opinion/experience added

- [ ] **Affiliate links added strategically**
  - 10-30 affiliate links total
  - Links in first 300 words
  - Links distributed throughout
  - Multiple links to top products
  - All use affiliate tag: shelzysdesigns-20

### Shortcodes & Elements
- [ ] **Product cards created**
  - Main products displayed in cards
  - Used `[product_card]` shortcode
  - Includes: ASIN, title, price, image, rating
  - Shows savings percentage if on sale

- [ ] **Comparison table added** (for list posts)
  - Used `[comparison_table]` shortcode
  - All products compared side-by-side
  - Includes: Name, price, rating, key features
  - Every row links to Amazon

- [ ] **Call-to-actions placed strategically**
  - CTA in introduction
  - CTA after each product/section
  - CTA at end of post
  - Varied button text:
    * "Check Price on Amazon"
    * "View Deal ‚Üí"
    * "Shop Now"
    * "See Current Price"

- [ ] **Deal badges added** (if applicable)
  - For time-sensitive deals
  - Used `[deal_badge]` shortcode
  - Creates urgency

### Engagement Elements
- [ ] **Email signup form added**
  - Used `[email_signup]` shortcode
  - Placed at end of post
  - Compelling offer/headline
  - Related to post topic

- [ ] **Social sharing buttons visible**
  - Pinterest, Facebook, Twitter
  - Placed at top and bottom of post

- [ ] **FAQ section added** (optional but recommended)
  - 3-5 common questions answered
  - Improves SEO (featured snippets)
  - Increases word count naturally

## SEO Optimization (Rank Math)

### Basic SEO
- [ ] **Focus keyword set**
  - Primary keyword: _____________
  - Used in title, URL, H2s, content
  - Rank Math score: 80+ / 100

- [ ] **URL optimized**
  - Short and clean
  - Includes target keyword
  - No dates (evergreen)
  - Example: `/best-amazon-beauty-dupes/`

- [ ] **Meta title written** (55-60 characters)
  - Includes target keyword
  - Compelling and click-worthy
  - Includes year (optional): "... 2024"
  - Example: "15 Best Amazon Beauty Dupes Under $25 (2024 Guide)"

- [ ] **Meta description written** (145-155 characters)
  - Includes target keyword
  - Includes call-to-action
  - Mentions benefit/value
  - Example: "Discover the best Amazon beauty dupes that look high-end but cost under $25. Save hundreds on luxury products! Shop now ‚Üí"

### Advanced SEO
- [ ] **Schema markup added**
  - Product schema for main products
  - Article schema for post
  - FAQ schema if applicable
  - Review schema if applicable

- [ ] **Internal links added**
  - Link to 2-3 related posts
  - Link from high-traffic posts to this post
  - Use descriptive anchor text
  - Don't use "click here"

- [ ] **External links added** (1-2 authoritative sources)
  - Link to relevant authority sites
  - Opens in new tab
  - Uses rel="nofollow" if needed

- [ ] **Image alt text optimized**
  - Every image has descriptive alt text
  - Includes keywords naturally
  - Describes image for accessibility

## Images & Media

### Featured Image
- [ ] **Featured image created**
  - Size: 1200x628px (Facebook share size)
  - Eye-catching and on-brand
  - Includes text overlay (optional)
  - Shows main product or concept
  - Compressed (under 100KB)

### Content Images
- [ ] **Product images added**
  - At least 1 image per product
  - High quality (but compressed)
  - Amazon product images (allowed)
  - Or royalty-free stock photos
  - Properly sized (max 800px wide)

- [ ] **Pinterest pin created**
  - Vertical format: 1000x1500px (2:3 ratio)
  - Eye-catching design
  - Uses brand colors: #FF6B6B, #FFE66D
  - Includes product image
  - Text overlay with key info
  - Watermark: "ShelzyPerkins.com"
  - Compressed (under 200KB)

- [ ] **All images optimized**
  - Compressed with ShortPixel
  - Under 100KB each (except featured/pin)
  - Format: JPEG for photos, PNG for graphics
  - Lazy loading enabled

---

## Mobile Optimization

- [ ] **Mobile preview checked**
  - Text readable on small screens
  - Buttons easily tappable (min 48x48px)
  - Images scale properly
  - No horizontal scrolling
  - Fast loading (under 3 seconds)

- [ ] **Touch-friendly elements**
  - CTA buttons large enough
  - Links have adequate spacing
  - Forms work on mobile
  - Menu accessible

---

## Legal & Compliance

- [ ] **Affiliate disclosure visible**
  - At top of post
  - Clear and conspicuous
  - Meets FTC guidelines
  - Can't be missed by readers

- [ ] **All links properly formatted**
  - Affiliate links include: rel="nofollow sponsored"
  - External links open in new tab
  - No broken links
  - Affiliate tag correct: shelzysdesigns-20

- [ ] **Product claims accurate**
  - No false claims
  - Prices accurate (or disclaimered)
  - Specifications verified
  - Personal opinions labeled as such

- [ ] **Copyright respected**
  - Only using authorized images
  - Amazon product images (allowed for affiliates)
  - Royalty-free stock photos with proper license
  - No copyrighted text copied

## Pre-Launch Testing

### Functionality Check
- [ ] **All affiliate links tested**
  - Click each link
  - Verify it goes to correct product
  - Check affiliate tag present
  - Test in incognito mode

- [ ] **Forms tested**
  - Email signup works
  - Confirmation email sent
  - No errors in console

- [ ] **Shortcodes rendered correctly**
  - Product cards display properly
  - Comparison tables work
  - Buttons styled correctly
  - No code visible on page

### Performance Check
- [ ] **Page speed tested**
  - Use: PageSpeed Insights
  - Desktop score: 90+
  - Mobile score: 80+
  - LCP under 2.5 seconds

- [ ] **Broken link check**
  - No 404 errors
  - All images load
  - External links valid

## Post-Launch Actions

### Immediate (Within 1 Hour)
- [ ] **Share on Pinterest**
  - Pin to relevant boards (3-5 boards)
  - Use optimized description
  - Add hashtags
  - Schedule additional pins

- [ ] **Share on social media**
  - Facebook page
  - Instagram story (if applicable)
  - Twitter (if active)

- [ ] **Submit to Google**
  - Request indexing in Search Console
  - Check that sitemap updated

### Within 24 Hours
- [ ] **Add internal links from old posts**
  - Edit 3-5 related old posts
  - Link to new post with keyword-rich anchor text

- [ ] **Monitor analytics**
  - Check initial traffic
  - Watch affiliate click tracking
  - Note any errors in console

- [ ] **Send in newsletter** (if weekly newsletter going out)
  - Feature as top post
  - Create compelling snippet

### Within 1 Week
- [ ] **Respond to comments**
  - Reply to all comments
  - Answer questions
  - Thank readers

- [ ] **Review performance**
  - Traffic sources
  - Time on page
  - Affiliate clicks
  - Email signups from post

- [ ] **Create additional Pinterest pins**
  - 3-5 different designs
  - Schedule over next 30 days

## 30-Day Follow-Up

- [ ] **Update with new products** (if applicable)
  - Check for better deals
  - Add newly launched products
  - Update prices if changed significantly

- [ ] **Check keyword rankings**
  - Use Google Search Console
  - Track position for target keyword
  - Adjust if not ranking well

- [ ] **Analyze affiliate performance**
  - Which products clicked most
  - Which sections got most engagement
  - Adjust layout if needed

- [ ] **Refresh if needed**
  - Update introduction
  - Add new section
  - Update images
  - Shows Google it's fresh content
